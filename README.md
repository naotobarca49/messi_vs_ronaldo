# サービス概要
長年続いたメッシ vs. C.ロナウド論争  
投票というシンプルな方法で終わりのない論争に終止符を打つ。  
<br>
# 登場人物
* エンドユーザー
  * メッシファン(信者)  
  * C.ロナウドファン(信者)
* 管理者
  * 投票者の情報を管理する人  
<br>
# ユーザーが抱える課題
メッシとC.ロナウドを様々な側面から分析してみても、両者ともに優れた実績があるため甲乙付け難い。  
<br>
# 解決方法
投票というシンプルな方法。  
どちらも優れた選手であるため論争などいらない。  
<br>
# プロダクト
メッシかC.ロナウドのどちらかに投票ができるWebアプリケーション。
